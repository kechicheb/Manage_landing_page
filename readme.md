# Usage

Install dependencies (Tailwind)

```
npm install
```

Run the Tailwind CLI to compile the **input.css** during development. The output file is **dist/main.css**

```
npm run watch
```

To build once run...

```
npm run build
```

You can edit the scripts in package.json and the tailwind.config.js file to change input/output locations
